Application Structure Guide

Panduan ini menjelaskan struktur aplikasi, serta fungsi dari setiap folder dan file yang terdapat di dalamnya.

  1. Home / Lobi
    Halaman utama (folder lobi/) tempat pengguna dapat mengakses fitur-fitur utama seperti navigasi, logout, dan tampilan lobi.
  2. Autentikasi dan Profil Pengguna
    • login/index.php: Form untuk masuk ke dalam aplikasi menggunakan email dan password.
    • login/register.php: Form pendaftaran pengguna baru.
    • profile/profile.php: Menampilkan informasi profil pengguna.
    • Fitur di dalam profil:
      • Nama Pengguna
      • Ubah Password
      • Kendaraan Saya
      • Privasi Akun
      • Pusat Bantuan
      • Jadwal Servis
      • Update tangkapan polisi
      • Logout (keluar dari akun)
  3. Fitur Utama
    • booking/index.php: Untuk pemesanan atau penjadwalan servis kendaraan.
    • activities/index.html: Menampilkan aktivitas atau riwayat layanan pengguna.
    • promo/fiturpromo.php: Menampilkan promo dan penawaran khusus yang tersedia.
  4. Kategori Produk
    • booking/: Meliputi data produk, oli, suku cadang, dan aksesoris.
    • booking/script.js: Logika interaktif dari halaman pemesanan dan produk.
  5. Manajemen Servis
    • jadwalservis/: Folder yang menyimpan pengelolaan jadwal servis kendaraan.
    • activities/riwayat.php: Menampilkan riwayat layanan yang sudah dilakukan pengguna.
  6. Pengaturan dan Personalisasi
    • login/script.js: Mengelola interaksi login dan registrasi.
    • profile/: Pengaturan tampilan dan data pengguna.
    • navbar/: Komponen navigasi antar halaman, memudahkan akses cepat ke berbagai bagian aplikasi.
    • navbar/koneksi.php: Koneksi ke basis data untuk navigasi.
  7. Fitur Tambahan
    • guide/guide.html: Halaman panduan aplikasi ini.
    • guide/jadwalservis: Panduan tambahan terkait servis kendaraan.
    • logout.php: File untuk keluar dari sesi pengguna.
    • coin (rencana): Sistem poin/reward yang mungkin akan diimplementasikan.

Panduan ini membantu dalam memahami tata letak proyek dan bagaimana setiap file berkontribusi terhadap fungsionalitas aplikasi.
